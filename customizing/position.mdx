---

Insetir texto descrevendo como funciona. Available position options: `topRight`, `topCenter`, `topLeft`, `right`, `center`, `left`, `bottomRight`, `bottomCenter`, `bottomLeft`.
```html index.html
<script>
    window.pactto = window.pactto || {};
    window.pactto.coach = 'username';
    window.pactto.company = 'Coach Name';
    // Add position attribute:
    window.pactto.position = 'bottomRight';
</script>
```
