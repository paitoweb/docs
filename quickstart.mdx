Copy the following JavaScript and paste it after the opening `<body>` tag.
```html index.html
<script src="https://pactto.com/assets/js/embed_api.js"></script>
```

Copy the following JavaScript and paste it before the closing `</body>` tag.
```html index.html
<script>
    window.pactto = window.pactto || {};
    window.pactto.coach = 'username';
    window.pactto.company = 'Coach Name';
</script>
```

Now just run the open function.
```html index.html
<button onclick="window.pactto.open()"></button>
```
